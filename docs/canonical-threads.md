# Canonical Thread Examples

Golden examples of the content Skygest should surface. Each entry demonstrates a different facet of expert energy discourse on Bluesky that no other platform aggregates.

## Categories

- **Data thread** — Original charts/data analysis with multiple images
- **Sourced thread** — Multi-post thread linking to primary sources
- **Expert discussion** — Post with substantive expert replies adding domain knowledge
- **Quote commentary** — Expert adding context to another expert's post
- **Breaking analysis** — Expert commentary on a developing story
